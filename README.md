# To-Do App

Esta es una aplicación web simple para gestionar tareas. Fue creada para aprender Azure DevOps.

## Características
- Agregar tareas.
- Marcar tareas como completadas.
- Eliminar tareas.

## Tecnologías
- Java SpringBoot
